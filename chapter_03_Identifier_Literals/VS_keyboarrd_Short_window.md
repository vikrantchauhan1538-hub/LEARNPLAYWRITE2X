# Visual Studio Code - Default Keyboard Shortcuts (Windows)

## General

| Shortcut | Action |
|:---|:---|
| `Ctrl + Shift + P` | Command Palette |
| `Ctrl + P` | Quick Open (Go to File) |
| `Ctrl + Shift + N` | New Window |
| `Ctrl + W` | Close Window |
| `Ctrl + ,` | Open User Settings |
| `Ctrl + K` `Ctrl + S` | Open Keyboard Shortcuts |
| `Ctrl + Shift + P` | Show Command Palette |
| `Ctrl + K` `Ctrl + R` | Open Keyboard Shortcuts Reference (PDF) |

| Shortcut | Action |
|:---|:---|

| `Ctrl + C` | Copy line (empty selection) |
| `Ctrl + V` | Paste |

| `Ctrl + Shift + Enter` | Insert Line Above |
| `Alt + Up Arrow` | Move Line Up |
| `Alt + Down Arrow` | Move Line Down |
| `Shift + Alt + Up Arrow` | Copy Line Up |
| `Shift + Alt + Down Arrow` | Copy Line Down |
| `Ctrl + /` | Toggle Line Comment |
| `Shift + Alt + A` | Toggle Block Comment |
| `Ctrl + D` | Add Selection to Next Find Match |
| `Ctrl + Shift + L` | Select All Occurrences of Current Selection |
| `Ctrl + F2` | Select All Occurrences of Current Word |
| `Ctrl + Shift + \` | Jump to Matching Bracket |
| `Ctrl + ]` | Indent Line |
| `Ctrl + [` | Outdent Line |
| `Ctrl + Home` | Go to Beginning of File |
| `Ctrl + End` | Go to End of File |
| `Ctrl + Up Arrow` | Scroll Line Up |
| `Ctrl + Down Arrow` | Scroll Line Down |
